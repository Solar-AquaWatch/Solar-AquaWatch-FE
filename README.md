## Data Sources and License Notice

Solar AquaWatch AX는 수로 수위 모니터링, 태양광 발전 가능량 예측, 위험 알림 기능을 위해 다음 데이터를 활용합니다.

### 공공데이터 API

1. 기상청_단기예보 조회서비스
- 제공처: 기상청 / 공공데이터포털
- 활용 목적: 날씨, 강수 여부, 구름 상태 기반 태양광 발전 가능량 예측
- 활용 필드: SKY, PTY, POP
- 서비스 내 활용 결과: solarPrediction, recommendedInterval

2. 한국수자원공사_우량수위 관측소 운영 정보
- 제공처: 한국수자원공사 / 공공데이터포털
- 활용 목적: 실제 우량·수위 관측 데이터 기반 위험도 보정
- 활용 필드: flux, hourwal, obsrdt
- 서비스 내 활용 결과: riskScore, reason, recommendedAction

본 저장소에는 공공데이터 API 인증키를 포함하지 않습니다.
API 인증키는 로컬 개발 환경의 ai_server/.env 파일에서만 관리합니다.

### AI Hub 데이터 사용 계획

본 프로젝트는 향후 모델 고도화를 위해 AI Hub 수위 측정 데이터 활용을 검토합니다.

- 데이터명: AI Hub 수위 측정 데이터
- 활용 목적: 수위표, 수면선, 수위 이미지 기반 딥러닝 모델 고도화
- 현재 MVP 활용 방식: OpenCV 기반 수위 분석 및 공공데이터 API 기반 위험도 산정
- 저장소 포함 여부: AI Hub 원본 이미지, 라벨 파일, 압축 파일, 모델 파일은 본 저장소에 포함하지 않음

AI Hub 원본 데이터는 제공기관의 이용정책을 준수하여 로컬 개발 환경에서만 사용하며, GitHub 등 외부 공개 저장소에 업로드하지 않습니다.

### 저작권 및 외부 자산 사용 주의

본 프로젝트는 대회 제출물 및 GitHub 공개를 고려하여 다음 원칙을 따릅니다.

- 외부 이미지, 브랜드 로고, 상용 아이콘, 유료 폰트는 무단 사용하지 않음
- 발표 자료와 시연 화면에는 직접 제작한 이미지, 직접 촬영한 이미지, 또는 사용 허가가 명확한 자료만 사용
- 공공데이터 사용 시 제공기관명과 데이터명을 명시
- API 인증키, 원본 데이터셋, 학습 데이터 원본은 저장소에 포함하지 않음

### Current AI Implementation

현재 MVP의 AI 서버는 다음 방식으로 동작합니다.

이미지 업로드 → OpenCV 기반 수위 추정 → 기상청 단기예보 API 기반 태양광 발전 가능량 예측 → K-water 우량·수위 관측 데이터 기반 위험도 보정 → Rule-based AI Engine으로 상태, 위험도, 판단 근거, 권장 조치 생성
